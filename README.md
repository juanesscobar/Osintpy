# OSIntPy

## Description
OSIntPy is an Open Source Intelligence (OSINT) tool designed to gather and analyze publicly available information from various sources.

## Features
- Multi-source data collection
- Information aggregation and analysis
- User-friendly interface
- Extensible architecture

## Installation
```bash
git clone https://github.com/juanesscobar/osintpy.git
cd osintpy
pip install -r requirements.txt
```

## Usage
```bash
python osintpy.py --help
```

## Requirements
- Python 3.8+
- See `requirements.txt` for dependencies
